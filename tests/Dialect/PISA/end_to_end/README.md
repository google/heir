# End to End PISA Tests

These tests use HEIR's ability to generate
[OpenFHE](https://github.com/openfheorg/openfhe-development) code for both or
client-side components such as key generation and encryption/decryption, and to
generate a ground-truth for the sever-side computation. They also produce the
polynomial/pISA code for the server-side computation to allow for testing and
benchmarking.
